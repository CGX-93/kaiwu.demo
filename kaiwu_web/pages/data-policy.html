TRANSPARENCY · 数据透明

你的数据，归你自己。

开物会记录你在站内的关键行为，用来改进方法论模型。这页讲清楚：我们收什么、不收什么、怎么用、你怎么拿回。这是一个开发中的占位说明，正式版会在下一阶段补全。

📥 我们会记录的

  • 你生成的工单（输入回答 + 输出 10 模块 + 你的反馈）— kaiwu_orders
  • 你查阅失败修复库的症状与方案效果（有用 / 部分有用 / 没用）— kaiwu_rescue_logs
  • 你提交的方法论原子（卖点 / 钩子 / 分镜 / 风格 / 修复）— kaiwu_atoms
  • 你提交的爆款解析记录（输入链接 / 描述 + 提取元素）— kaiwu_decode_records
  • 关键页面访问 / 点击事件，用于优化体验 — kaiwu_events
  • 一个仅本机生成的匿名设备 ID — kaiwu_meta.deviceId

🚫 我们不会收的

  • 不收任何账号 / 实名 / 手机号 / 邮箱
  • 不读取你电脑里其他网站的内容
  • 不和广告商共享
  • 不用作训练数据之外的任何用途

🎯 用来做什么

  • 训练 5 个领域专项模型：卖点提炼 / 钩子开头 / 分镜节奏 / 风格锁定词 / 失败修复
  • 统计哪些模板最常被采用、哪些急救方案真正有效
  • 识别工单输出里需要改进的环节（你点 👎 的模块）

🛡️ 你的权利

  • 查：所有数据都存在你的浏览器 localStorage 里，你随时可在历史页一键查看
  • 导出：一键下载完整 JSON，自己留底或带走
  • 删除：随时清空全部数据，删完即彻底消失
  • 现阶段未上传任何数据到服务器；未来上线后端时，会再次明确通知
